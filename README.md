#مشروع مسار للتصميم الداخلي والديكور
## م|وسيلة نصر عبد الرقيب
###فكرة المشروع:
موقع الكتروني لشركة باسم مسار متخصصة في الهندسة المعمارية والتصميم الداخلي
####الادوات المستخدمة
HTML(لهيكلة المحتوى)
CSS(لتصميم الواجهة)
Javascript(اضافة تفاعل حي)
#####المميزات
تصميم انيق يتناسب مع مجال الديكورات
واجهة تواصل تفاعلية
تجربة مسنخدم سلسة
